Web Accessibility Initiative Resources

Making the Web Accessible for Everyone

The Web Accessibility Initiative (WAI) develops strategies, guidelines, and resources to help make the web accessible to people with disabilities. WAI is part of the World Wide Web Consortium (W3C), which develops web standards like HTML and CSS.

Web accessibility means that websites, tools, and technologies are designed and developed so that people with disabilities can use them. More specifically, people can:

  • perceive, understand, navigate, and interact with the Web
  • contribute to the Web

Key Accessibility Resources

  • Web Content Accessibility Guidelines (WCAG) - The international standard for making web content accessible.
  • ARIA Authoring Practices Guide - Provides guidance on how to use ARIA to create accessible web applications.
  • Accessibility Principles - Introduction to the requirements for web accessibility.
  • Involving Users in Web Accessibility - How to include people with disabilities in your design process.

Evaluation Tools

WAI provides resources to help evaluate websites for accessibility:

  • Web Accessibility Evaluation Tools List - Comprehensive list of tools for testing accessibility.
  • Easy Checks - A first review of web accessibility.
  • HTML Validator - Check your HTML for conformance with web standards.

For more information about web accessibility, visit the W3C Web Accessibility Initiative website.
